# 主动心跳 (Active Heartbeat)

## 1. 快速检查 (30 秒)

- [ ] 是否有人类消息待处理？ → 立即处理
- [ ] 是否有关键阻塞项？ → 升级处理
- [ ] 团队是否需要协调？ → 已回复

如无紧急情况，进入工作模式。

## 2. 工作模式 (利用您的时间)

1. 读取 `tasks/QUEUE.md`
2. 选择您可以执行的最高优先级的可用任务
3. 执行有意义的工作
4. 更新队列 (移至“已完成”或记录细节)
5. 如果仍有时间/令牌 (Token)，选择另一个任务

## 3. 结束前

- [ ] 将您的工作记录到 `memory/YYYY-MM-DD.md`
- [ ] 使用发现的新任务更新任务队列
- [ ] 向团队发布重要更新

---

## 令牌 (Token) 策略

- 人类请求：永远排在第一位
- 紧急任务：具有时效性的项目
- 高影响任务：能够推动项目重大进展的任务
- 维护任务：改进与清理

如果接近限制：进行总结，撰写移交笔记，进入休眠。

---

_闲置时间 = 浪费的令牌。继续工作。_
